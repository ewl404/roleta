# Aurora Cartomante - Mapa da Sorte

Este é um protótipo de alta fidelidade para a "Aurora Cartomante", apresentando o "Mapa da Sorte".

## Tecnologias Utilizadas

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS** para estilização luxuosa
- **Lucide React** para ícones
- **Genkit** para sugestões de IA (Roleta)
- **ShadCN UI** para componentes de interface

## Estrutura do Projeto

- `/roleta`: O jogo da roleta mística.
- `/carta`: O jogo de cartas para ganhar giros.
- `/resgatar`: O fluxo de resgate de prêmio via PIX com verificação de segurança.

## Comandos para Deploy (Vercel/Git)

Para preparar o projeto para deploy, você pode usar os seguintes comandos:

```bash
# 1. Inicialize o Git (se ainda não tiver)
git init

# 2. Adicione todos os arquivos
git add .

# 3. Faça o primeiro commit
git commit -m "Primeiro commit - deploy vercel"
```
